1. Теоретические основы методов анализа данных

1.1. Метод наименьших квадратов (МНК)

Метод наименьших квадратов (МНК) — это один из наиболее распространенных и фундаментальных методов в математической статистике и регрессионном анализе. Его основная задача — найти оптимальные параметры математической модели, которая наилучшим образом описывает зависимость между переменными на основе имеющихся данных. Ключевая идея МНК заключается в минимизации суммы квадратов отклонений (остатков) между наблюдаемыми (фактическими) значениями зависимой переменной и значениями, предсказанными выбранной моделью. Это означает, что метод ищет такую функцию (например, прямую линию в случае линейной регрессии), для которой сумма квадратов вертикальных расстояний от каждой точки данных до этой функции является минимальной. Именно это "наименьшее" количество квадратов и дало методу его имя. Для простой линейной регрессии, представляемой уравнением \(\(y=a+bx\)\), параметры a и b вычисляются по следующим формулам: - Коэффициент наклона (b): \(\(b = (n * \Sigma X Y - \Sigma X * Sigma Y) / (n * Sigma X^2 - (\Sigma X)^2)\)\) - Свободный член (a): \(\(a = (\Sigma Y - b * \Sigma X) / n\)\) Здесь n представляет количество точек данных, ΣX — сумму всех значений X, ΣY — сумму всех значений Y, ΣXY — сумму произведений X и Y для каждой точки, и ΣX^2 — сумму квадратов значений X. МНК может быть использован для определения линейной или полиномиальной зависимости между переменными. А ещё он хорошо работает в зашумлённых данных. Метод наименьших квадратов, за счет минимизации суммы квадратов ошибок, эффективно усредняет влияние случайного шума в данных. Важным ограничением МНК является его чувствительность к выбросам (аномальным значениям) в данных, которые могут существенно исказить результаты модели. Это означает, что для надежного применения МНК крайне важен этап предварительной обработки данных.

1.2. Метод моментов (ММ)

Метод моментов (ММ) — это один из старейших и наиболее интуитивно понятных методов оценки неизвестных параметров распределений в математической статистике и эконометрике. Он был предложен Карлом Пирсоном в 1894 году. Основная идея ММ заключается в замене теоретических моментов (например, математического ожидания, дисперсии) исследуемого распределения их выборочными аналогами (эмпирическими моментами), которые вычисляются непосредственно из наблюдаемых данных. Если распределение имеет k неизвестных параметров, то для их оценки необходимо составить систему из k уравнений, приравняв первые k выборочных моментов к соответствующим теоретическим моментам. Решение этой системы уравнений дает оценки параметров. ММ используется для получения оценок параметров распределений, когда аналитические формулы для теоретических моментов известны и относительно просты. Например, для оценки среднего значения (μ) и дисперсии (σ²) нормального распределения, можно приравнять выборочное среднее к μ, а выборочную дисперсию к σ².

1.3. Метод максимального правдоподобия (ММП)

Метод максимального правдоподобия (ММП) — это один из наиболее мощных и широко используемых статистических методов для оценки параметров модели. Его основная идея заключается в нахождении таких значений параметров, которые делают наблюдаемые данные наиболее "правдоподобными" или вероятными. Центральным понятием является функция правдоподобия L(параметры | данные), которая выражает вероятность наблюдения данного набора данных при заданных значениях параметров модели. Цель ММП — найти параметры, максимизирующие эту функцию. Для упрощения математических вычислений, особенно при дифференцировании, часто используется логарифм от функции правдоподобия. Поскольку логарифм является монотонной функцией, максимизация логарифма функции правдоподобия эквивалентна максимизации самой функции правдоподобия, но при этом произведение вероятностей преобразуется в сумму, что значительно упрощает дальнейшие шаги. ММП применяется в широком спектре статистических моделей, включая линейные и обобщенные линейные модели, факторный анализ, моделирование структурных уравнений. Он также используется в задачах проверки статистических гипотез и построения доверительных интервалов. Широко применяется в различных областях, таких как биостатистика, эконометрика, машинное обучение и др.

1.4. Корреляционный анализ

Корреляционный анализ — это статистический метод, предназначенный для измерения силы и направления статистической связи (взаимосвязи) между двумя или более переменными. Коэффициент корреляции, как правило, принимает значения в диапазоне от -1 до +1. Значение +1 указывает на сильную прямую (положительную) связь, -1 — на сильную обратную (отрицательную) связь, а 0 — на отсутствие линейной связи. Применение корреляционного анализа включает оценку взаимосвязей между различными показателями в экономике, маркетинге (например, между маркетинговыми инвестициями и их результатами) , а также идентификацию потенциальных взаимосвязей для дальнейшего, более глубокого анализа. Для корректного применения параметрических методов корреляции (например, коэффициента Пирсона) необходимо, чтобы совокупность значений признаков подчинялась многомерному нормальному распределению. Важно помнить, что наличие высокой корреляции между двумя переменными (X и Y) не дает основания утверждать, что одна переменная является причиной изменения другой. Существует несколько причин, по которым корреляция может наблюдаться без причинно-следственной связи: обе переменные могут быть вызваны третьим, неучтенным фактором; связь может быть случайной; или направление причинности может быть обратным (например, не маркетинговые расходы увеличивают продажи, а рост продаж позволяет увеличить маркетинговый бюджет.

1.4.1. Ранговый коэффициент корреляции Спирмена

Коэффициент ранговой корреляции Спирмена (обозначается как ρ или rs) — это непараметрическая мера монотонной связи между двумя переменными. Он оценивает, насколько хорошо отношение между двумя переменными может быть описано монотонной функцией (то есть, если одна переменная увеличивается, другая также имеет тенденцию к увеличению, но не обязательно линейно). Поскольку он основан на рангах данных, а не на их фактических значениях, он устойчив к выбросам и не требует предположения о нормальности распределения данных. Этапы расчета коэффициента Спирмена включают: 1. Присвоение рангов (как по возрастанию, так и по убыванию). Каждому значению присваивается ранг. 2. Определение разностей рангов (d): Для каждой пары наблюдений вычисляется разность между их рангами по двум переменным (\(\(d = R(Xi) - R(Yi)\)\)). 3. Возведение в квадрат и суммирование: Каждая полученная разность d возводится в квадрат, и затем все квадраты суммируются (\(\(\sigma d^2\)\)). 4. Вычисление коэффициента по формуле: Если все ранги являются различными целыми числами (нет связанных рангов): \(\(\rho = 1 - (6 * \Sigma d^2i) / (n * (n^2 − 1))\)\), где n — число наблюдений. При наличии связанных рангов используется более сложная формула, которая учитывает поправки на связанные ранги, или же можно использовать формулу Пирсона для ранжированных данных.

Ключевое преимущество ранговых коэффициентов корреляции, таких как Спирмен, заключается в их непараметрической природе. Это означает, что они не требуют предположений о нормальности распределения данных, что часто является проблемой в реальных наборах данных. Более того, они могут быть применены к порядковым (качественным) данным, где значения имеют естественный порядок, но не являются числовыми в строгом смысле (например, "низкий", "средний", "высокий").

1.4.2. Коэффициент ранговой корреляции Кендалла Тау

Коэффициент ранговой корреляции Кендалла Тау (τ) — это еще одна непараметрическая мера монотонной связи, которая является альтернативой коэффициенту Спирмена и часто предпочтительнее для малых размеров выборок. Метод основан на анализе пар наблюдений и подсчете количества согласованных и несогласованных пар. - Согласованная пара (nC): Пара наблюдений (x1, y1) и (x2, y2) считается согласованной, если \(\(sgn(x2-x1) = sgn(y2-y1)\)\). То есть, если x2 > x1 и y2 > y1, или x2 < x1 и y2 < y1. - Несогласованная пара (nD): Пара наблюдений считается несогласованной, если \(\(sgn(x2-x1) = -sgn(y2-y1)\)\). То есть, если x2 > x1 и y2 < y1, или x2 < x1 и y2 > y1. Формула для тау-а (τA), базового типа, без связанных рангов : \(\(\tau_A = (nC - nD) / (0.5 \cdot n \cdot (n - 1))\)\) или \(\(\tau_A = 1 - (4 \cdot nD) / (n \cdot (n - 1))\)\). Существуют модификации коэффициента Кендалла, такие как τB и τC, которые учитывают наличие связанных рангов (одинаковых значений). Тау-б, в частности, часто используется в бизнес-аналитике и имеет поправки на связанные ранги.

Построить по признакам X и Y интервальные вариационные ряды, количество интервалов определить по формуле Стерджеса. Построить полигон и гистограмму для каждого из признаков.

Для рассчёта количества интервалов по формуле Стерджеса используется следующая формула: \(\(k = 1 + 3.322 \cdot \log_{10}(n)\)\) где n — количество наблюдений в выборке. Эта формула позволяет определить оптимальное количество интервалов для построения гистограммы, чтобы избежать избыточной детализации или потери информации. Для построения интервальных рядов и гистограмм в Python мы будем использовать библиотеки Pandas и Matplotlib.

import numpy as np
import matplotlib.pyplot as plt

# у нас есть столбцы "№", "X" и "Y"
n = len(df)
k = 1 + 3.322 * np.log10(n)  # Формула Стерджеса для количества интервалов
k = int(round(k))

print(f"Количество интервалов по Стерджесу: {k}")

# Интервальный ряд для X
range_X = df['X'].max() - df['X'].min()
interval_width_X = range_X / k

# Создание гистограммы для X
bins_X = np.arange(df['X'].min(), df['X'].max() + interval_width_X, interval_width_X)
counts_X, bin_edges_X = np.histogram(df['X'], bins=bins_X)
bin_centers_X = (bin_edges_X[:-1] + bin_edges_X[1:]) / 2

interval_series_X = pd.DataFrame({
    'Interval': [f'[{bin_edges_X[i]:.2f}, {bin_edges_X[i+1]:.2f})' for i in range(len(bin_edges_X)-1)],
    'Frequency': counts_X,
    'Relative Frequency': counts_X / n
})
hprint(interval_series_X, "interval_X")

# Построение полигона и гистограммы для X
# Примечание! В pyodide графики отображаются в самом низу страницы, а не внизу вывода.
plt.figure(figsize=(10, 6))
plt.hist(df['X'], bins=bins_X, edgecolor='black', alpha=0.7)
plt.title('Histogram for Feature X')
plt.xlabel('X Values')
plt.ylabel('Frequency')
plt.grid(axis='y', alpha=0.75)
plt.show()

plt.figure(figsize=(10, 6))
plt.plot(bin_centers_X, counts_X, marker='o', linestyle='-', color='blue')
plt.title('Frequency Polygon for Feature X')
plt.xlabel('X Values (Midpoints of Intervals)')
plt.ylabel('Frequency')
plt.grid(True)
plt.show()

# Интервальный ряд для Y
range_Y = df['Y'].max() - df['Y'].min()
interval_width_Y = range_Y / k
bins_Y = np.arange(df['Y'].min(), df['Y'].max() + interval_width_Y, interval_width_Y)
counts_Y, bin_edges_Y = np.histogram(df['Y'], bins=bins_Y)

2. Оценить статистические характеристики рядов распределения.

2.1. Показатели центра распределения: средняя, мода, медиана. 2.2. Показатели колеблемости признака: размах вариации, дисперсия, среднее квадратическое отклонение, коэффициент вариации, коэффициент асимметрии, коэффициент эксцесса.

Для оценки статистических характеристик рядов распределения в Python можно использовать методы библиотеки Pandas. Мы будем рассчитывать следующие показатели:

# Расчет статистических характеристик для признака X
mean_X = df['X'].mean()  # Среднее
median_X = df['X'].median()  # Медиана
mode_X = df['X'].mode()[0]  # Мода
range_X = df['X'].max() - df['X'].min()  # Размах вариации
variance_X = df['X'].var(ddof=1) # ddof=1 для выборочной дисперсии
std_dev_X = df['X'].std(ddof=1) # ddof=1 для выборочного стандартного отклонения
coeff_variation_X = (std_dev_X / mean_X) * 100 # в процентах
skewness_X = df['X'].skew()
kurtosis_X = df['X'].kurt()

# Расчет статистических характеристик для признака Y
mean_Y = df['Y'].mean()  # Среднее
median_Y = df['Y'].median()  # Медиана
mode_Y = df['Y'].mode()[0]  # Мода
range_Y = df['Y'].max() - df['Y'].min()  # Размах вариации
variance_Y = df['Y'].var(ddof=1) # ddof=1 для выборочной дисперсии
std_dev_Y = df['Y'].std(ddof=1) # ddof=1 для выборочного стандартного отклонения
coeff_variation_Y = (std_dev_Y / mean_Y) * 100 # в процентах
skewness_Y = df['Y'].skew()
kurtosis_Y = df['Y'].kurt()

# Создание DataFrame для статистических характеристик признака X и y
stats = pd.DataFrame({
    'Statistic': ['Mean', 'Median', 'Mode', 'Range', 'Variance', 'Std Dev', 'Coeff of Variation (%)', 'Skewness', 'Kurtosis'],
    'X': [mean_X, median_X, mode_X, range_X, variance_X, std_dev_X, coeff_variation_X, skewness_X, kurtosis_X],
    'Y': [mean_Y, median_Y, mode_Y, range_Y, variance_Y, std_dev_Y, coeff_variation_Y, skewness_Y, kurtosis_Y]
})
hprint(stats, "stats")

3. На основе одного из критериев согласия Пирсона проверить гипотезу о законе нормального распределения.

Для проверки гипотезы о нормальности распределения признаков X и Y можно использовать критерий согласия Пирсона (χ²). Этот критерий позволяет сравнить наблюдаемое распределение данных с теоретическим нормальным распределением. Для этого необходимо: 1. Разделить данные на интервалы (как мы уже сделали ранее). 2. Рассчитать ожидаемое количество наблюдений в каждом интервале, исходя из нормального распределения с параметрами, полученными ранее (среднее и стандартное отклонение). 3. Вычислить χ²-статистику по формуле:

from scipy.stats import chisquare, norm
# Функция для расчета ожидаемых частот на основе нормального распределения
def expected_frequencies(data, bins):
    mean = data.mean()
    std_dev = data.std(ddof=1)
    expected_freq = []
    total_prob = 0
    n = len(data)

    # Сначала вычисляем все вероятности
    for i in range(len(bins) - 1):
        lower_bound = bins[i]
        upper_bound = bins[i + 1]
        prob = norm.cdf(upper_bound, loc=mean, scale=std_dev) - norm.cdf(lower_bound, loc=mean, scale=std_dev)
        expected_freq.append(prob * n)
        total_prob += prob

    # Нормализуем частоты, чтобы сумма совпадала с наблюдаемой
    expected_freq = [freq * (n / (total_prob * n)) for freq in expected_freq]
    return expected_freq

# Расчет ожидаемых частот для признака X
expected_freq_X = expected_frequencies(df['X'], bin_edges_X)

# Расчет ожидаемых частот для признака Y
expected_freq_Y = expected_frequencies(df['Y'], bin_edges_Y)
# Расчет χ²-статистики для признака X
chi2_X, p_X = chisquare(counts_X, f_exp=expected_freq_X)
# Расчет χ²-статистики для признака Y
chi2_Y, p_Y = chisquare(counts_Y, f_exp=expected_freq_Y)
# Создание DataFrame для результатов теста
results = pd.DataFrame({
    'Feature': ['X', 'Y'],
    'Chi-squared': [chi2_X, chi2_Y],
    'p-value': [p_X, p_Y]
})
hprint(results, "chi2_results")

4. Для параметров X и Y определить тесноту связи с помощью коэффициента корреляции.

Для оценки тесноты связи между признаками X и Y можно использовать коэффициент корреляции Пирсона. Этот коэффициент измеряет линейную зависимость между двумя переменными и принимает значения от -1 до +1, где: - +1 указывает на идеальную положительную линейную зависимость, - -1 указывает на идеальную отрицательную линейную зависимость, - 0 указывает на отсутствие линейной зависимости.

5. Построить линейную функцию регрессии, найти коэффициент эластичности.

Для построения линейной функции регрессии между признаками X и Y можно использовать метод наименьших квадратов. Линейная функция регрессии имеет вид: \(\(Y = a + bX\)\) где a — свободный член (пересечение с осью Y), b — коэффициент наклона.

from sklearn.linear_model import LinearRegression

# Создание модели линейной регрессии
model = LinearRegression()

b = model.coef_[0]

elasticity = b * (df['X'].mean() / df['Y'].mean())
# Создание DataFrame для результатов регрессии
regression_results = pd.DataFrame({
    'Coefficient': ['Intercept (a)', 'Slope (b)', 'Elasticity'],
    'Value': [a, b, elasticity]
})
